# Vocabulário do "Revele seu Hobby"

Este documento define os termos-chave utilizados no ecossistema do aplicativo "Revele seu Hobby". Com o objetivo de manter um padrão e entendimento unificado e guiar o desenvolvimento e design.

|       Termo        | Definição                                                                                                                                            | Exemplo de Uso / Contexto                                                                     |
| :----------------: | ---------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
|     Comunidade     | Um espaço virtual dedicado a um hobby específico. É o principal local de interação, onde os membros podem fazer posts, comentar e organizar eventos. | "Entrei para a **Comunidade** de 'Churrasco em casa' para tirar umas dúvidas."                |
| Comunidade Pública | Uma comunidade de acesso livre, onde qualquer usuário pode entrar e visualizar o conteúdo sem a necessidade de aprovação.                            | "Workshop de desenho, ensinando iniciantes"                                                   |
| Comunidade Privada | Uma comunidade restrita que exige a aprovação do Moderador para a entrada de novos membros. O conteúdo é visível apenas para quem está dentro.       | "Para discutir os detalhes do projeto, criamos uma **Comunidade Privada**."                   |
|       Evento       | Uma atividade organizada com data, hora e local definidos, relacionada a um hobby e uma comunidade.                                                  | "Vamos criar um **Evento** para o nosso encontro mensal de jogos de tabuleiro."               |
|   Evento Público   | Um evento aberto para a inscrição de qualquer usuário do aplicativo, mesmo que não faça parte da comunidade organizadora.                            | "A palestra sobre 'Introdução à Robótica' será um **Evento Público**."                        |
|   Evento Privado   | Um evento exclusivo para membros de uma comunidade específica ou para usuários convidados, que necessitam de aprovação para participar.              | "A confraternização será um **Evento Privado**, apenas para os membros ativos da comunidade." |
|       Hobby        | A atividade ou interesse que serve como tema central para as Comunidades e Eventos. Os hobbies são "tags" que conectam usuários **(Hobbistas)**      | "Adicione o **Hobby** 'Pintura a Óleo' no seu perfil para receber recomendações."             |
|      Hobbista      | Nome dados ao Usuário do aplicativo                                                                                                                  | ----                                                                                          |
|       Perfil       | A página pessoal de cada usuário, contendo informações como nome, foto, lista de hobbies e as comunidades das quais faz parte.                       | "Visitei o **Perfil** dele e vi que também temos o hobby de 'Ciclismo' em comum."             |
| Post / Publicação  | Qualquer conteúdo (texto, foto, vídeo, link) compartilhado por um membro no feed de uma Comunidade.                                                  | "Ela fez um **Post** incrível mostrando o progresso do seu projeto de marcenaria."            |
|     Moderador      | O criador e administrador de uma Comunidade. Ele é responsável por aprovar novos membros (em comunidades privadas) e gerenciar o conteúdo postado.   | "O **Moderador** removeu um post que não estava de acordo com as regras."                     |
